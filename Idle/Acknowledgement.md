# Focus 100

## Updatelog

### 1.1 (23)

* 调整了时间过渡动画的样式

### 1.1 (22)

* 新增：为 macOS 14 以上的机型增加了时间的过渡动画

### 1.1 (21)

* 新增：这台设备上的专注时间将展示在主界面
* 优化了部分界面的显示效果

### 1.1 (19)

* 优化了部分界面的显示效果

### 1.0 (17)

* 精简了项目体积

### 1.0 (16)

* 修复了一些潜在的漏洞

### 1.0 (15)

* 修复了在其他语言环境下可能导致程序无法启动的问题

### 1.0 (14)

* 对于部分窗口下的显示效果进行了调整

### 1.0 (10)

* 修复了在弹出窗口时，窗口可能不会处于焦点的问题

* 修复了重复点击按钮时，重复打开窗口的问题

已知问题：

* 在连接不通畅的网络中进行同步可能引发用户信息丢失的问题
